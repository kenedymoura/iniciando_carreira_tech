## 🚀 Ordem Recomendada
1. **Python (Bradesco)** → consolidar lógica de programação.  
2. **SQL Server (Cursa/FGV)** → aprender manipulação de dados.  
3. **JavaScript (Trybe/Udemy)** → iniciar projetos web.  
4. **C# (Microsoft/Udemy)** → ampliar para aplicações robustas.  
5. **GitHub** → documentar e publicar todos os projetos.
